 This assignment is about solving the well known {\bfseries{2D lid-\/driven cavity incompressible flow}} problem using different algorithms and schemes.





Lid-\/driven cavity incompressible flow solution

\hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysection{\texorpdfstring{Problem statement}{Problem statement}}\label{md__r_e_a_d_m_e_autotoc_md1}
The problem consists in a square cavity with a lid moving at a constant velocity. The flow is incompressible and the Reynolds number is low. {\bfseries{Navier-\/\+Stokes}} equations and the {\bfseries{pressure-\/velocity coupling}} are used to solve the problem.\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysection{\texorpdfstring{Methods and schemes}{Methods and schemes}}\label{md__r_e_a_d_m_e_autotoc_md2}
The code is able to solve the problem using various methods and schemes. In particular, we can choose the following options\+:


\begin{DoxyItemize}
\item Methods\+:
\begin{DoxyItemize}
\item {\bfseries{Gauss-\/\+Siedel (SCGS)}}
\item {\bfseries{SIMPLE}}
\end{DoxyItemize}
\item Schemes\+:
\begin{DoxyItemize}
\item {\bfseries{Convection schemes}}\+:
\begin{DoxyItemize}
\item UDS
\item Hybrid
\item QUICK
\end{DoxyItemize}
\item {\bfseries{Diffusion schemes}}\+:
\begin{DoxyItemize}
\item Second order central difference
\item Fourth order central difference
\end{DoxyItemize}
\end{DoxyItemize}
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md3}{}\doxysection{\texorpdfstring{Data in}{Data in}}\label{md__r_e_a_d_m_e_autotoc_md3}
